# ADR 0004: Use USWDS Design Principles

## Status

Accepted

## Context

The project is intended to demonstrate readiness for government contractor and civic technology software engineering roles.

## Decision

Use USWDS-inspired design principles for layout, accessibility, forms, navigation, and content hierarchy.

## Consequences

### Positive

- Better accessibility practices
- More government-aligned design approach
- Stronger portfolio relevance for public sector roles

### Negative

- Requires learning USWDS patterns
- Some USWDS patterns may need adaptation for React
